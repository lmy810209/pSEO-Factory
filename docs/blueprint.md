# **App Name**: GenSite AI

## Core Features:

- Topic & Requirements Input: User provides a central theme or topic for the website, along with any other specific requirements or keywords for SEO.
- AI Content & Structure Generation: An AI tool powered by Claude API dynamically generates SEO-optimized content, including page text, meta descriptions, titles, and a suggested multi-page site structure based on the input topic.
- Dynamic Next.js Site Assembly: Automatically assembles the generated content and structure into a functional Next.js application, including page components and routing.
- Vercel Deployment Orchestration: Initiates the deployment of the generated Next.js project to Vercel via the Vercel API upon user confirmation.
- Subdomain Configuration: Automates the assignment of a unique subdomain under 'igeol.kr' to the newly deployed site.
- Deployment Status & History: Users can view the real-time deployment status of their projects and access a historical list of all generated sites with their live URLs.

## Style Guidelines:

- Primary color: A vibrant yet sophisticated digital blue (#20A0E4) to convey efficiency and innovation.
- Background color: A subtle, light blue-tinted white (#ECF3F7) for a clean and unobtrusive backdrop.
- Accent color: A dynamic orange (#EB872F) for key calls to action and important highlights, providing energetic contrast.
- Headline and body font: 'Inter' (sans-serif) for its modern, clear, and objective aesthetic, ideal for a functional web application.
- Utilize minimalist line icons for clarity and to align with the modern, digital aesthetic, focusing on concepts like AI, deployment, and content generation.
- A clean, intuitive single-page application layout with generous whitespace, ensuring a streamlined user experience from input to deployment monitoring. Prioritize responsive design for optimal viewing across devices.
- Subtle loading spinners and transition animations to provide clear feedback during site generation and deployment processes, enhancing perceived responsiveness without distracting.
